# Blackjack / 21

Um jogo simples de terminal feito em PHP e C++ baseado no Blackjack / 21.

O jogador compra cartas tentando chegar o mais próximo possível de 21 sem ultrapassar esse valor. O dealer joga automaticamente após o turno do jogador.

## Exemplo

```txt id="34khbq"
1 - Seguir
2 - Parar
1
PLAYER: 10

1 - Seguir
2 - Parar
1
PLAYER: 17

1 - Seguir
2 - Parar
2
DEALER: 20
PLAYER perdeu
```

## Como executar
Recomendo executar pelo VSCode (caso já esteja configurado), Dev-C++ no caso do C++, ou utilizar compiladores online como o [Online C Compiler](https://www.onlinegdb.com/online_c_compiler) ou o [OnlinePHP](https://onlinephp.io)
